## Installation

Install the package globally with the JS package manager of your choice:

<Tabs>
  <TabItem label="npm">
        ```
        npm install -g xs-dev
        ```
  </TabItem>
  <TabItem label="pnpm">
        ```
        pnpm install -g xs-dev
        ```
  </TabItem>
  <TabItem label="yarn">
        ```
        yarn global add xs-dev
        ```
  </TabItem>
</Tabs>
